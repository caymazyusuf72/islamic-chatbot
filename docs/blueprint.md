# **App Name**: NurAI

## Core Features:

- AI-Powered Answers: Receive user questions and use Google Gemini 2.0, as a tool, to provide answers grounded in the Quran, Hadith, and Islamic scholarship.
- Referenced Responses: Display AI-generated answers with clear references to the Quran (Surah and verse), Hadith (source), or scholarly opinions.
- Theme Switching: Implement a light/dark theme toggle for user preference.
- API Key Protection: Securely store the Google Gemini 2.0 API key in a .env file.
- Animated Greeting: Implement an animated "Bismillahirrahmanirrahim" greeting upon initial app load.

## Style Guidelines:

- Primary color: Soft gold (#C2B280) evoking reverence and warmth.
- Background color: Very light beige (#F5F5DC), offering a sense of calm.
- Accent color: Muted olive green (#808000) to highlight key information and interactive elements.
- Headline font: 'Lateef' (serif) for titles, and 'PT Sans' (sans-serif) for body text.
- Maintain a clean and minimalist design, emphasizing whitespace to avoid distractions.
- Incorporate subtle animations for a modern and engaging user experience.
